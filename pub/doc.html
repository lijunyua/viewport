Getting started

Download the Viewport folder, ensure both Viewport.js and style.css are in it.
Place the folder at the same level of you HTML file.
Use standard HTML and load the Viewport library:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Viewport Demo</title> <script defer type="text/javascript" src='./Viewport/Viewport.js'></script> <script defer type="text/javascript" src='./your_javascript_file.js'></script> <link rel="stylesheet" type="text/css" href="your_stylesheet.css"> </head> <body> </body> </html>

You can modify img src below so it finds some images on your machine.
Add this code to your HTML body:

<div id="myViewList"> <a href="https://www.google.com"> <img class="myView" src="./image/random-pic.jpg"> </a> <img class="myView" src="./image/random-pic1.jpeg"> <img class="myView" src="./image/random-pic2.jpg"> <img class="myView" src="./image/random-pic3.jpg"> <img class="myView" src="./image/random-pic4.jpg"> <div class="myView"> Lorem ipsum dolor sit amet, consectetur adipiscing elit. Ut semper justo ut venenatis maximus. Cras vestibulum urna dolor, ut aliquet ex vehicula nec. Morbi dictum est non arcu rutrum efficitur. Integer in quam tincidunt, accumsan neque ac, varius libero. Proin aliquam nisi ante, vitae ultricies lorem laoreet nec. Nam ut eros vulputate, pulvinar ipsum et, molestie libero. Curabitur nec faucibus odio. Donec convallis vestibulum volutpat. </div> </div>

And this to your_javascript_file.js:

"use strict"; const viewport = new Viewport() const myViewList = document.querySelector("#myViewList") const allViews = document.querySelectorAll(".myView") viewport.makeView(myViewList, allViews)

Refresh your HTML page and now you should see small blocks of your images and textboxes arranged.

API Methods

viewport.makeView(viewList, allViews, dimensions, setViewStyle, setHighlightStyle, setSelectStyle) 
	  Creates Wrapper divs that wrap elements in allViews inside viewList and style these wrappers.

	  viewList:          Required HTML element. 
	                     Represents the element such that every element in it that is included in allViews would be styled.

	  allViews:          Required HTML collection.
	                     Represents the elements to be styled.

	  dimensions:        Optional 2-dimensional array of integers.
	                     Represents the defined width and height of each element.
	                     Expected length is equal to the length of allViews.
	                     Expected length of each sub-array is 2.
	                     Default: 2-dimensional array with each sub-array [10, 10], with length equal to the length of allViews.

	  setViewStyle:      Optional function which takes one parameter wrapper.
	                     Applies this style to all elements in allViews inside viewList when no mouseover/mousedown occur.
	                     Default:
	                     function _setDefaultViewStyle(wrapper) {
	                         wrapper.style.borderWidth = '8px'
	                         wrapper.style.borderStyle = 'solid'
	                         wrapper.style.borderColor = 'white'
	                     }

	  setHighlightStyle: Optional function which takes one parameter wrapper.
	                     Applies this style to all elements in allViews inside viewList when no mouseover occur.
	                     Default:
	                         function _setDefaultViewStyle(wrapper) {
	                             wrapper.style.borderColor = 'yellow'
	                         }

	  setSelectStyle:    Optional function which takes one parameter wrapper.
	                     Applies this style to all elements in allViews inside viewList when mousedown occur.
	                     Default:
	                         function _setDefaultViewStyle(wrapper) {
	                             wrapper.style.borderColor = 'green'
	                         }
viewport.makeViewClosable(viewList, closableViewList) 
	  Call after makeView. Adds a close button to wrapper of each element in closableViewList inside viewList.

	  viewList:          Required HTML element. 
	                     Expected to be one that was used to call makeView successfully.

	  closableViewList:  Required HTML collection.
	                     Represents the elements to add a close button.
viewport.makeViewReorderable(viewList, mode) 
	  Call after makeView. Adds reordering functionality to 

	  viewList:          Required HTML element.
	                     Expected to be one that was used to call makeView successfully.

	  mode:              Optional String.
	                     Expected value is 'reorder' or 'swap'.
	                     Default value is 'reorder'.
	                     Represents the reordering method to use.
viewport.setUpSaveAndResetButtons(viewList, buttons) 
	  Call after makeView and makeViewReorderable. Adds save and reset functionality to elements in buttons.

	  viewList:          Required HTML element.
	                     Expected to be one that was used to call makeView and makeViewReorderable successfully.

	  buttons:           Required 2-dimensional array of HTML elements.
	                     Represents the buttons to add functionality to.
	                     Expected length is equal to the length of allViews.
	                     Expected length of each sub-array is 2.
	                     In each subarray, the first button will be added save functionality.
	                     In each subarray, the second button will be added reset functionality.
viewport.setUpResetDefaultButton(viewList, button) 
	  Call after makeView and makeViewReorderable. Adds reset default view functionality to button.

	  viewList:          Required HTML element.
	                     Expected to be one that was used to call makeView and makeViewReorderable successfully.

	  buttons:           Required HTML element.
	                     Represents the button to add functionality to.
viewport.setUpChangeOrderingMethodButton(viewList, button) 
	  Call after makeView and makeViewReorderable. Adds switch ordering method functionality to button.

	  viewList:          Required HTML element.
	                     Expected to be one that was used to call makeView and makeViewReorderable successfully.

	  buttons:           Required HTML element.
	                     Represents the button to add functionality to.
